Control Flow

Python provides various control flow statements, including if-else statements and loops.

Conditional statements allow the execution of different code blocks based on certain conditions.

if x > 5: print("x is greater than 5") else: print("x is less than or equal to 5")

Loops, such as for and while loops, enable repetitive execution of code.
